# Моё тело

12-недельная программа осознанных отношений с едой, телом и мышлением. Работа с пищевым поведением и РПП через физиологию, психологию и поведение.

**[→ открыть приложение](https://elenamirzoeva.github.io/TELO/)**

---

## Что внутри

**Авторизация** — регистрация и вход по email + пароль. Данные хранятся в облаке (Supabase), не пропадут при смене браузера или устройства.

**Дашборд** — приветствие, текущая неделя, статистика (записи, прогресс чек-листа, streak дневника), блок «с чего начать» с вводными по неделе, задачи на сегодня, фокус недели, цитата дня.

**Программа 12 недель** — три блока по 3 недели + неделя интеграции. Каждая неделя: теория + практика. Три столпа: **физиология** → **психология** → **поведение**, каждый виток глубже.

- Недели 1–3: знакомство с голодом, карта привычек, эмоции и еда
- Неделя 4: интеграция
- Недели 5–7: типы переедания, тревога/стыд, движение как радость
- Неделя 8: интеграция
- Недели 9–11: нейрофизиология стресса, образ тела, устойчивые привычки
- Неделя 12: финал

**Дневник** — записи с шкалой голода/насыщения (1–10), 18 эмоций на выбор, тип голода (физический / эмоциональный / социальный / по привычке), заметки.

**Чек-листы** — задания на каждую из 12 недель. Навигация, прогресс-бар, сохранение состояния.

**Трекеры** — графики голода/насыщения за 7 дней, карта эмоций, календарь записей, streak, прогресс всех чек-листов.

**Ресурсы** — книги (Intuitive Eating, «Тело хранит счёт», Self-Compassion), DBT-техники (STOP, TIPP), практики (шкала голода, body scan, дыхание 4–7–8, заземление 5–4–3–2–1), наука (set point, дофаминовая петля, гарвардская тарелка).

## Стек

- Vanilla HTML / CSS / JS — без фреймворков и сборщиков
- [Supabase](https://supabase.com) — авторизация (email + пароль) и хранение данных
- localStorage — кэш для быстрой загрузки
- GitHub Pages — хостинг

---

## Опора на источники

- Intuitive Eating (Tribole & Resch)
- DBT-навыки для эмоциональной регуляции
- ACT для работы с образом тела
- Самосострадание (Kristin Neff)
- Нейробиология привычек (Duhigg, Fogg)
- MBSR (Kabat-Zinn)

## Запуск локально

```bash
python3 -m http.server 8787
```

Открыть `http://localhost:8787`. Для работы авторизации нужен настроенный Supabase-проект (см. `supabase-setup.sql`).
