# ТЕЛО — программа осознанных отношений с собой

Персональная веб-программа на 12 недель: работа с телом, мышлением, пищевым поведением и РПП. Открой `index.html` в браузере или запусти локальный сервер и открой `http://localhost:8787`.

---

## Что внутри

**Дашборд** — приветствие по имени, текущая неделя, статистика (записи, прогресс чек-листа, streak дневника), задачи на сегодня, фокус недели, цитата дня.

**Программа 12 недель** — три блока по 3 недели + неделя интеграции. Каждая неделя раскрывается: теория + практическое упражнение. Три столпа: **физиология** → **психология** → **поведение**, каждый виток глубже.

- Недели 1–3: знакомство с голодом, карта привычек, эмоции и еда  
- Неделя 4: интеграция  
- Недели 5–7: типы переедания, тревога/стыд, движение как радость  
- Неделя 8: интеграция  
- Недели 9–11: нейрофизиология стресса, образ тела, устойчивые привычки  
- Неделя 12: финал  

**Дневник** — записи с шкалой голода/насыщения (1–10), 18 эмоций на выбор, тип голода (физический / эмоциональный / социальный / по привычке), заметки. Всё сохраняется в `localStorage`.

**Чек-листы** — для каждой из 12 недель свой набор заданий. Навигация по неделям, прогресс-бар, чекбоксы с сохранением состояния.

**Трекеры** — графики голода/насыщения за 7 дней, карта частых эмоций, календарь записей, streak дневника, прогресс всех 12 чек-листов.

**Ресурсы** — книги (Intuitive Eating, «Тело хранит счёт», Self-Compassion), DBT-техники (STOP, TIPP), практики (шкала голода, body scan, дыхание 4–7–8, заземление 5–4–3–2–1), научные концепции (set point, дофаминовая петля, гарвардская тарелка).

## Опора на источники

- Intuitive Eating (Tribole & Resch)  
- DBT-навыки для эмоциональной регуляции  
- ACT для работы с образом тела  
- Самосострадание (Kristin Neff)  
- Нейробиология привычек (Duhigg, Fogg)  
- MBSR (Kabat-Zinn)

## Запуск

- Открыть `index.html` в браузере (Chrome, Safari, Firefox и т.д.).  
- Или из папки проекта: `python3 -m http.server 8787` и перейти на `http://localhost:8787`.

Файлы: `index.html` + `app.js`. Без зависимостей, работает офлайн, данные хранятся в браузере (`localStorage`).
